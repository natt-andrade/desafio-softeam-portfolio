# Portfólio - Desafio Softeam

Este projeto é um site portfólio desenvolvido como parte do processo seletivo da Empresa Júnior de Computação da Universidade Federal de Sergipe (UFS) **Softeam**. O objetivo foi criar uma aplicação web seguindo as boas práticas de desenvolvimento da empresa, com foco em fluxo de trabalho com Git (Git Flow) e padronização de commits.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização responsiva com Flexbox e identidade visual.
* **JavaScript:** Manipulação de DOM para atualização automática do ano no rodapé.
* **Git & GitHub:** Versionamento de código e hospedagem.

## Funcionalidades

* Apresentação pessoal e profissional.
* Listagem de projetos com links diretos para demonstração e repositório.
* Design responsivo (adaptável para mobile e desktop).
* Navegação para redes sociais (LinkedIn e GitHub).
